## üîç How It Works

code-review-agent scans your code in the markdown file by looking into 14 categories such as:

- Code style and formatting  
- Security issues, based on OWASP guidelines  
- Potential bugs and incorrect logic  
- Use of best coding practices  
- Performance suggestions  

It organizes issues by severity:

- Level 1: Informational tips  
- Level 2: Minor issues to consider  
- Level 3: Serious warnings that should be fixed  
- Level 4: Critical problems needing immediate action  

The tool runs static analysis, which means it looks at the code without running it. It can also fix some issues automatically which saves you time.
